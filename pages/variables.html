Introdução

Variáveis são nomes simbólicos que vão receber algum valor.

O JavaScript é uma linguagem fracamente tipada e dinâmica, quer dizer que não precisamos especificar um tipo para nossa variável, quando você atribui um valor a ela, por baixo do panos o JavaScript defini automaticamente o tipo que ela será, podendo mais para frente, mudar o seu tipo, diferente de outras linguagens que isso não é possível.

Criando variáveis

Para criar uma variável, exitem 3 maneiras, são elas: var, let e const, cada uma tem um comportamento diferente que iremos abordar.

Sintaxe

Primeiro declaramos uma palavra reservada para criar uma variável, usaremos a palavra let, depois escolha um nome que coincide com o valor que será atribuido, logo em seguida atribuimos o valor com o sinal ( = ).

                  
                  var foo = 'valor';
                  let foo = 'valor';
                  const foo = 'valor';

Note que usamos o operador de igualdade ( = ), mas neste caso esse operador tem a função de atribuição, ele está atribuindo um valor a variável de nome 'foo', você pode usar qualquer nome pra uma variável, e quando invocar a mesma, ela retornará o que está dentro dela, no caso o valor que foi atribuido a ela.

                  
                  let fruta = 'Banana';
                  console.log(fruta);
                  Banana

Não é necessário colocar o nome da variável em aspas, por você não quer a palavra fruta, e sim o que está dentro da variável fruta, que será o valor atribuido, podendo ser uma string, number, array, objeto e etc. Se não souber nada disso, de uma olhada em Tipos de Dados.

Scope

O escopo determina a visibilidade de uma variável no JavaScript

O escopo é determinado pelos colchetes ( {} ), tudo o que estiver dentro do colchetes, é um bloco de código

                  
                  {
                     let fruta = 'Banana';
                     console.log(fruta);
                     Banana
                  }

O escopo de um bloco, é tudo o que está dentro dos colchetes.

Var x Let x Const

Global
É tudo o que está executando na minha aplicação, em todo o seu arquivo .js
Local
É somente o que existe dentro de um escopo

Var

A palavra reservada var, defini uma variável com um escopo global, e local, podendo funcionar fora do escopo de um bloco

O JavaScript cria a variável var no começo de sua execução, e só depois atribui um valor a ela

                     
                     {
                        let animal = 'Jabuti';
                        console.log(animal);
                        Jabuti
                     }
                     console.log(animal);
                     Jabuti

Let e Const

Let e const são variáveis locais, e só funcionam dentro do escopo em que foram criadas.

                     
                     {
                        let animal = 'Tartaruga';
                        console.log(animal);
                        Tartaruga
                     }
                     console.log(animal)
                     *Isso irá resulta em um erro de referência, 
                     em que animal não foi denifido*

Outra característica do const, é que não pode ser reatribuido um valor

                     
                     {
                        const fruta = 'Laranja';
                        fruta = 'Maça';
                        *Resultará um um erro*
                     }

Nomenado variáveis

O JavaScript é casa-sensitive(sensível ao caso), e aceita a cadeia de caracteres Unicode.

  • Permitido
    • Iniciar com estes caractes especiais: $ e _
    • Iniciar com letras
    • Colocar acentos
    • Letras minúsculas e maiúsculas, fazem a diferença
  • Não é permitido
    • Iniciar com números
    • Colocar espaços vazios no nome
  • Dicas
    • Criar nomes que fazem sentido
